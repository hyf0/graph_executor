# graph-executor

An multithreaded computation scheduler. Execute your tasks in parallel with based on dependencies.
# Features

- Synced API
  - [ ] circle detection
  - [ ] Parallel execution
  - [ ] Concurrency Limiter
  - [ ] Priority

- Asynced API 
  - [ ] circle detection
  - [x] Parallel execution
  - [x] Concurrency Limiter
  - [x] Priority
