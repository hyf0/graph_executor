# graph-executor

An multithreaded computation scheduler. Execute your tasks in parallel with based on dependencies.
# Features

- Synced API
  - [x] Parallel execution
  - [ ] Concurrency Limiter
  - [ ] Priority
- Asynced API 
  - [ ] Parallel execution
  - [ ] Concurrency Limiter
  - [ ] Priority

Inspired by [p-graph](https://github.com/microsoft/p-graph)
